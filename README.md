# **AMD-FX-Hackintosh-Opencore-EFI**

## MacOS Catalina 10.15.7
## Bootloader: OpenCore 0.6.6

## My PC specs: 
- CPU: 
  - Amd FX-8320
- GPU: 
  - Sapphire r9 280 3gb
- Ram: 
  - 2 x 4 Kingston HyperX Blu @1600
- Motherboard:
  - AsRock 990fx extreme 3
- HDD:
  - Kingston 320gb
